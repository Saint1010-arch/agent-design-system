# Examples — Agent 设计示例

本目录将收录使用五层架构模板设计的完整 Agent 配置示例。

## 即将推出

- **产品经理 Agent** — 战略规划 + 需求分析 + 竞品对标
- **招聘专家 Agent** — 简历筛选 + 面试设计 + 人才评估

## 贡献示例

欢迎提交你用本项目模板设计的 Agent 配置！请确保：

1. 使用 `templates/五层架构模板.md` 完成全部五层
2. 通过 `templates/质检清单.md` 检查
3. 方法论标注来源
4. 以 Pull Request 形式提交到 `examples/<角色名>/` 目录
